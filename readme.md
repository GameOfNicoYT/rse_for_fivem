# RSE Command

## Installation:

1. Put the Folder in the resources Folder
2. Add `ensure rse` to the `server.cfg`

## Usage:

- `/rse start` : Start the recording
- `/rse stop` : Stops the recording
- `/rse open` : Opens the Rockstar Editor
